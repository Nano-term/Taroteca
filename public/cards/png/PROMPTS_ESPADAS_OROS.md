# Prompts completos · Espadas y Oros (28 cartas faltantes)

Cada prompt de abajo ya está listo para copiar y pegar en el generador de
imágenes (Midjourney, DALL-E, etc.). Incluye: estilo base + acento de palo +
sujeto de la carta + instrucción de no incluir texto.

## Especificaciones técnicas

| Parámetro | Valor |
|---|---|
| Formato | JPG (calidad 80 a 90) |
| Proporción | **7:12** (vertical, proporción de carta de tarot) |
| Resolución recomendada | **840 × 1440 px** (mínimo 560 × 960 px) |
| Peso objetivo | < 250 KB por carta (optimiza con [squoosh.app](https://squoosh.app)) |
| Carpeta destino | `/public/cards/espadas/` y `/public/cards/oros/` |
| Nombre de archivo | exactamente el slug indicado en cada prompt |

Genera las 28 en la misma sesión o con la misma semilla/modelo para mantener
la consistencia visual del mazo.
